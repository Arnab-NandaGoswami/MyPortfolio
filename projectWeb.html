Shopify

Technology Used :-
HTML, CSS, Bootstrap, PHP, XAMP

Here in this Project I've made a fully funtional E-Commerce Website named Shopify that can be assumed as a best alternative of the Amazon or Flipkart. It can be best described as a Amazon Clone or Flipkart Clone where for making the skeleton structure, I've used HTML, for Designing purpose, I've used CSS & Bootstrap and for making it fully functional and to integrate the website to the database, I've used the PHP as a scripting language to establish a secure connection between the site and the database. In the Local server that is created by XAMP, The website is tasted to be fullu funtional. New Users can get themselves in by simplying registering themselves using SignUp form and exixting users can also logging into their account using Login form.
